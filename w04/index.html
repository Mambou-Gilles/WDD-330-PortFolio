Form Events:

The focus event occurs when an element is focused on.
Example: input.addEventListener('focus', () => alert('focused'), false);

The blur event occurs when the user moves the focus away from the form element.
Example: input.addEventListener('blur', () => alert('blurredd'), false);  

  

The change event occurs when the user moves the focus away from the form element after changing it.
Example: input.addEventListener('change', () => alert('changed'), false);

The most important form event is the submit event, occurring when the form is submitted.
Example: const form = document.forms['search'];
     form.addEventListener ('submit', search, false);
     function search(event) {
      alert('Form Submitted');
       event.preventDefault();
      }
In this instance we stopped the form from being submitted to the URL by using the preventDefault() method
