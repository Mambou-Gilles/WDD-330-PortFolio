The JavaScript object is a collection of key-value pairs, similar to a map, dictionary, or hash-table in other programming languages. Anything that is not a JS primitive is an Object.

  • An Object is a collection of properties.
  • A Property is a key-value pair that contains a name and a value.
  • A Property Name is a unique value that can be coerced to a string that points to a value.
  • A Property Value can be any value, including other objects or functions, that associated with the name/key.

JavaScript uses the keywords const and let to declare variables.
The keyword const is used when the variable will not be reassigned to another value, whereas let is used if the variable might be reassigned later in the program.

There are two types of scope in JavaScript.These are:

  • Global scope — Global scope contains all of the things defined outside of all code blocks. A code block simply consists of grouped statements inside curly braces ({ }). if statement, loops, function are examples of structures that create a code block.
    A global variable has global scope. A global variable is accessible from anywhere in the code.
  • Local Scope — Local scope contains things defined inside code blocks. A local variable has local scope. A local variable is only accessible where it’s declared.

When writing your codes it is good to use functions for reasons such as:

  • To improve the readability of code.
  • Improves the reusability of the code, the same function can be used in any program rather than writing the same code from scratch.
  • Debugging of the code would be easier if you use functions, as errors are easy to be traced.

The two most used functions are function declaration and function expression
Function declaration example:
function hello(){ console.log('Hello World!');}

Function expression example:
const goodbye = function(){ console.log('Goodbye World!'); };

There is another syntax used to declare a function and it is called arrow. These make declaring functions much more succinct by using less verbose syntax. Arrow functions can be identified by the 'arrow' symbol, => that gives them their name
